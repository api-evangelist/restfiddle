# RESTFiddleThis is a repo for managing machine readable artifacts for the [RESTFiddle APIs](http://www.restfiddle.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [RESTFiddle APIs](http://www.restfiddle.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.restfiddle.com/)- [Developer](http://www.restfiddle.com/)- [Twitter](https://twitter.com/restfiddle)- [GitHub](https://github.com/AnujaK/restfiddle)The [APIs.json](https://github.com/api-evangelist/restfiddle/blob/master/apis.json) index provides links to all the relevant links for the RESTFiddle APIs, including the OpenAPI and Postman if they are present.
